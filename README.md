# 策征时刻 CeZhengShiKe
率土之滨 · 游戏专用闹钟与计划提醒工具

## 项目介绍
《策征时刻》是一款专为 **率土之滨** 玩家打造的移动端工具类 APP，
帮助玩家在游戏赛季中**更准时、更清晰、更省心**地完成关键操作。

## 主要功能
- 赛季计划、任务、闹钟提醒
- 率土地图城池/坐标参考与记录
- 自定义提醒时间与内容
- 赛季数据导入、配置管理
- 简洁清爽的本地界面，无广告、无后台偷跑

## 适用人群
- 率土之滨等slg类游戏玩家
- 需要固定时间提醒开荒、开关卡、打城的玩家
- 征服赛季、S1–S3 通用

## 技术说明
- 开发语言：Kotlin
- 框架：Android 原生
- 架构：简洁本地存储
- 特点：轻量、无权限滥用、本地运行、不开网络上传隐私

## 开源说明
本项目开源仅用于学习和方便率土玩家使用，
**不涉及游戏破解、不修改游戏数据、不影响游戏平衡**。

所有代码遵循开源精神，可学习、可二次修改，
但请勿用于商业用途或恶意打包。
